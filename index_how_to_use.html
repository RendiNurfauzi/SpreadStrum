SPREAD SPECTRUM IMAGE ENCODER AND DECODER
--------HOW TO USE--------
INI ADALAH PROGRAM UNTUK MEMASUKAN PESAN RAHASIA PADA GAMBAR PROGRAM INI MENGGUNAKAN ALGHORITMA SPREAD SPECTRUM YANG DAPAT MENJAGA KERAHASIAN PESAN ANDA SECARA OPTIMAL DAN RAHASIA ! ! ! !! ! !
----------ENCODE----------

1. UPLOAD GAMBAR ANDA DENGAN MENEKAN TOMBOL “ UPLOAD IMAGE “

2. INPUTKAN PESAN ANDA PADA TABEL INPUT MESSAGE

3. TEKAN TOMBOL ENCODE UNTUK MEMASUKAN PESAN ANDA PADA GAMBAR

4. SETELAH BERHASIL, MAKA REPRESENTASI PESAN ANDA DALAM BENTUK BINARY AKAN DI TAMPILKAN PADA KOLOM “ BINARY REPRESENTATION “

5. DOWNLOAD GAMBAR ANDA UNTUK MENYIMPAN NYA

----------DECODE----------

1. UPLOAD GAMBAR ANDA DENGAN MENEKAN TOMBOL “ UPLOAD IMAGE “

2. TEKAN TOMBOL DECODE UNTUK MENDAPATKAN PESAN ANDA PADA GAMBAR

3. SETELAH BERHASIL, MAKA REPRESENTASI PESAN ANDA DALAM BENTUK BINARY AKAN DI TAMPILKAN PADA KOLOM “ BINARY REPRESENTATION “

4. PESAN RAHASIA AKAN DI TAMPILKAN PADA KOLOM OUTPUT MESSAGE

JAGALAH KERAHASIAAN PESAN RAHASIA ANDA !!!
